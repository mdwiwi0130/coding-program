寫一個程式來判斷你給的數，落在什麼區域？

如果是正數，首先印出「（此數）是正數」，
小於等於10000，則顯示「A」；
其餘正數，則顯示「B」。

負數，則印出「（此數）是負數」，
小於等於-10000，則顯示「C」；
其餘負數，則顯示「D」。

因為此題輸出有多種狀況，故需將程式碼複製3次，都判斷正確才給分，每個輸出都要換行。

輸入範例:
6000
-99
50000

輸出範例:
6000是正數
A
-99是負數
D
50000是正數
B
```
/*10_3.比較中有比較-玩玩二分法(30.0)*/
#include <iostream>     
using namespace std; 
int main() { 
    int a,z=1;
    while (z<=3){
        cin>>a;
        if(a>=0){
            cout<<a<<"是正數"<<endl;
            if(a<=10000)
                cout<<"A"<<endl;
            else
                cout<<"B"<<endl;
        }
        else{
            cout<<a<<"是負數"<<endl;
            if(a<=-10000)
                cout<<"C"<<endl;
            else
                cout<<"D"<<endl;
            
        }
    ++z;
    }
    
    return 0; 
}
```
