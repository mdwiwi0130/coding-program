題目
```
寫一個程式，在記憶體放置一個整數3，然後印出來。

輸入範例:


輸出範例:3
```
程式範例
```
C範例程式:
#include<stdio.h>
int main()
{
int num;
num=3;
printf("......\n",......);
return 0;
}

C++範例程式:
#include<iostream>
using namespace std;
int main()
{
int num;
num=3;
cout<<num<<endl;
return 0;
}
```
解法
  ```
  /*3_1.資料也有家-印出指定的整數(15.0)*/
#include <iostream>     
using namespace std; 
int main() { 
    int a=3;
    cout<<a;

    return 0; 
}
  ```
