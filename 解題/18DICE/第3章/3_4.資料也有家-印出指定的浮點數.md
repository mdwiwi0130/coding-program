題目
```
 寫一個程式，指定二個浮點數為10.100000與5.200000，並且印出結果，格式如輸出範例所示。

輸入範例:

輸出範例:
num1=10.100000
num2=5.200000
```
解法
```
 /*3_4.資料也有家-印出指定的浮點數(15.0)*/
#include <iostream>     
using namespace std; 
int main() { 
    float num1=10.1,num2=5.2;
    cout<<"num1="<<num1<<"00000"<<endl;
    cout<<"num2="<<num2<<"00000";

    return 0; 
}
```
