題目
```
 寫一個程式，指定一個整數為160，一個浮點數為55.22，分別代表身高與體重，並印出其值，格式如輸出範例所示。

輸入範例:

輸出範例:
My height is 160 CM.
My weight is 55.22 KG.
```
解法
```
/*3_5.資料也有家-身高與體重(15.0)*/
#include <iostream>     
using namespace std; 
int main() { 
    int a=160;
    float b=55.22;
     cout<<"My height is "<<a<<" CM."<<endl;
     cout<<"My weight is "<<b<<" KG.";


    return 0; 
}
```
