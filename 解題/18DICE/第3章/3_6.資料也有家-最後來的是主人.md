題目
```
 寫一個程式，先指定一個整數變數為1，然後指定它為2，再指定它為3，再指定它為4，再指定它為5，印出變數最後的值。

程式碼範例如下：
a=1;
a=2;
a=3;
a=4;
a=5;

輸入範例:

輸出範例:
5
```
解法
```
/*3_6.資料也有家-最後來的是主人(15.0)*/
#include <iostream>     
using namespace std; 
int main() { 
    int a=1;
     a=2;
     a=3;
     a=4;
     a=5;
     cout<<a;
    return 0; 
}
```
