題目
```
寫一個程式，印出如下的資料，同行的資料間以\t間隔:

輸入範例:

輸出範例:

1	1	1
2	4	8
3	9	27
4	16	64
5	25	125
6	36	216
7	49	343
8	64	512
9	81	729
10	100	1000
```
解法
```
 /*2_5.讓螢幕會做秀-來個表格(20.0)*/
#include <iostream>     
using namespace std; 
int main() { 
    int a=0,n=0,w=0;
    while(a<=9){
    a++;
    n=a*a;
    w=a*n;
    cout<<a<<"\t"<<n<<"\t"<<w<<endl;
}
    return 0; 
}
```
