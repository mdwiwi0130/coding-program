題目
```
寫一個程式，印出如以下的結果：

輸入範例:

輸出範例:
******
*****
****
***
**
*

```
解法
```
/*2_3.讓螢幕會做秀-倒印三角形(20.0)*/
#include <iostream>     
using namespace std; 
int main() { 
    cout<<"******"<<endl;
        cout<<"*****"<<endl;
        cout<<"****"<<endl;
        cout<<"***"<<endl;
        cout<<"**"<<endl;
        cout<<"*"<<endl;

    return 0; 
}
```
