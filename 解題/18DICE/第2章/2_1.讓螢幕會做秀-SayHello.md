寫一個程式，印出Hello, DICE!

輸入範例:

輸出範例:
Hello, DICE!

C範例程式:
#include<stdio.h>
int main()
{
    printf("Hello, DICE!\n");
    return 0;
}

C++範例程式：
#include <iostream>
using namespace std;

int main()
{
cout<<"Hello, DICE!\n";/*cout 是輸出函式*/
return 0;
}
