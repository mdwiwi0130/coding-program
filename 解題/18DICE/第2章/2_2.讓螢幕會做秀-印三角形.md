題目:
```
寫一個程式，印出以下結果：

輸入範例:

輸出範例:
*
**
***
****
*****
******
```
解法:
```
/*2_2.讓螢幕會做秀-印三角形(10.0)*/
#include <iostream>     
    using namespace std; 
    int main() 
    {
        cout<<"*"<<endl;
        cout<<"**"<<endl;
        cout<<"***"<<endl;
        cout<<"****"<<endl;
        cout<<"*****"<<endl;
        cout<<"******"<<endl;
        return 0; 
    }
```
